The <b>inverse</b> of a linear transformation $f: V \to W$ is the linear transformation $f^{-1}: W \to V$ such that $f \circ f ^{-1} = id$ and $f^{-1} \circ f = id$

A transformation is called <b>invertible</b> if it has an inverse

<u>Theorem 17.2</u>
A linear transformation $f: V \to W$ has an inverse if and only if $f$ is both injective and surjective(or is a bijection)

An isomorphism is an invertible linear transformation. Two vector spaces $V$ and $W$ are isomorphic, denoted $V \cong W$, if there is an isomorphism between them

<u>Theorem 17.4</u>
Two finite-dimensional vector spaces are isomorphic if and only if they have the same dimension

<u>Theorem 17.5: Proving that a Linear Transformation is an Isomorphism</u>
Let $V$ and $W$ be finite-dimensional vector spaces. A linear transformation $f: V \to W$ is an isomorphism if we have two of the following three properties:
	$f$ is injective
	$f$ is surjective
	$V$ and $W$ have the same dimension
