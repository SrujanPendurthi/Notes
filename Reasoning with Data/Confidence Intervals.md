Inference - the process of using a sampling distribution to give a probability that a sample statistic of interest might take certain values by chance

There exists two branches of Formal Inference:
	Confidence Interval - A set of the most-plausible values for an unknown parameter
	Hypothesis Testing - A way to judge the probability of a presumed value for a parameter(aka Significance Testing)

The mean is called a point estimate, but it is an imperfect estimate

Confidence Interval - a set of most-likely values of an unknown parameter. A spread of numbers on the numberline which we hope covers the parameter
				$$\bar{X} \pm Z_{critical}\frac{\sigma}{\sqrt{n}}$$ the margin of error is the term on the right of the $\pm$, which does not account for bias in non-random sampling

If a supposed parameter value lies inside a confidence interval, then the supposition is a reasonable one

If a supposed parameter value lies outside a confidence interval, then the supposition is a not reasonable one

There is an inverse relationship between precision and confidence, without choosing a larger sample size

Confidence Interval for population proportion(valid if Binomial)
$$\hat{p} \pm Z_{critical}\sqrt{\frac{(\hat{p})(1-\hat{p})}{n}}$$

Use the minus for left endpoint and plus for right endpoint
